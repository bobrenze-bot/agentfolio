# Profile Mirroring to Decentralized Storage

**Date:** 2026-03-05

## Options

### IPFS + Filecoin
- **Best for:** General data, profiles
- **Pros:** Widely used, Filecoin storage deals
- **Cons:** Needs pinning service

### Arweave
- **Best for:** Permanent records
- **Pros:** One-time payment, permanent
- **Cons:** More expensive upfront

### Ceramic (StreamRegistry)
- **Best for:** Mutable data
- **Pros:** Updateable, DID-based
- **Cons:** Complex setup

## Recommendation

For AgentFolio profiles:
1. **Primary:** Keep on Cloudflare Pages
2. **Backup:** IPFS with Filecoin
3. **Permanent archive:** Arweave
